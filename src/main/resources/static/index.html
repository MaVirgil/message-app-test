HTML + CSS TEST SIDE

Denne side udgør mit første egentlige Spring Boot webprojekt, hvilket medfører at en stor del af den bagomliggende kode ikke har noget funktionelt grundlag, men blot eksisterer som øvelse af grundlæggende koncepter inden for Spring Boot

Tjek sidens kode ud på Github!

Sidens opbygning

Denne side består af et simpelt HTML-dokument, som linker til et css stylesheet. HTML-dokumentet består af simple elementer med semantiske tags, såsom header (<h1>) og paragraph (<p>). Denne tekst befinder sig i HTML-dokumentets <body>, som generelt indeholder størstedelen af sidens indhold.
